## 1. 快速开始

### 最小可运行示例

```cpp
#include <nged/nged.h>
#include <nged/nged_imgui.h>
#include <nged/entry/entry.h>

class MyApp : public nged::App {
  nged::EditorPtr editor;
  char const* title() override { return "我的编辑器"; }

  void init() override {
    App::init();  // 设置暗色主题
    editor = nged::newImGuiNodeGraphEditor();
    editor->initCommands();
    editor->createNewDocAndDefaultViews();
  }

  void update() override {
    static auto prev = std::chrono::system_clock::now();
    auto now  = std::chrono::system_clock::now();
    auto dt   = std::chrono::duration<float>(now - prev);
    editor->update(dt.count());
    editor->draw();
    prev = now;
  }
};

int main() {
  nged::startApp(new MyApp());
}
```

运行后你会得到一个空白的节点图编辑器窗口，可以右键创建节点、拖拽连线。

## 2. 创建第一个节点

节点是图的基本单元。继承 `nged::Node` 并重写引脚数量和绘制方法：

```cpp
class AddNode : public nged::Node {
public:
  AddNode(nged::Graph* parent)
    : nged::Node(parent, "add", "加法") {}  // 类型标识, 显示名称

  nged::sint numMaxInputs() const override { return 2; }   // 2 个输入
  nged::sint numOutputs()  const override { return 1; }    // 1 个输出
};
```

### 引脚说明

| 方法 | 返回值 | 说明 |
|------|--------|------|
| `numMaxInputs()` | `sint` | 最大输入数，`< 0` 表示不限制 |
| `numFixedInputs()` | `sint` | 固定输入数（仅不限输入时有效） |
| `numOutputs()` | `sint` | 输出引脚数 |
| `acceptInput(port, src, srcPort)` | `bool` | 是否接受连接（类型检查） |

## 3. 定义节点工厂

节点工厂告诉编辑器如何创建节点和列出可用类型：

```cpp
class MyNodeFactory : public nged::NodeFactory {
  // 创建根图
  nged::GraphPtr createRootGraph(nged::NodeGraphDoc* doc) const override {
    return std::make_shared<nged::Graph>(doc, nullptr, "root");
  }

  // 根据类型名创建节点
  nged::NodePtr createNode(nged::Graph* parent, std::string_view type) const override {
    if (type == "add")
      return std::make_shared<AddNode>(parent);
    return nullptr;
  }

  // 列出可用节点类型（供右键菜单和 Ctrl+P）
  void listNodeTypes(
      nged::Graph* parent, void* ctx,
      void(*cb)(void*, nged::StringView category, nged::StringView type, nged::StringView name))
    const override
  {
    cb(ctx, "数学", "add", "加法");       // 分类, 类型标识, 显示名称
    cb(ctx, "数学", "mul", "乘法");
    cb(ctx, "输入输出", "in", "输入");
    cb(ctx, "输入输出", "out", "输出");
  }
};
```

## 4. 组装编辑器

在 `App::init()` 中配置编辑器：

```cpp
void init() override {
  App::init();

  editor = nged::newImGuiNodeGraphEditor();

  // 1. 节点工厂 — 定义可创建的节点
  editor->setNodeFactory(std::make_shared<MyNodeFactory>());

  // 2. 图元工厂 — 定义可创建的图元（注释框、箭头、路由等）
  editor->setItemFactory(nged::addImGuiItems(nged::defaultGraphItemFactory()));

  // 3. 视图工厂 — 定义视图类型（网络视图、检查器等）
  editor->setViewFactory(nged::defaultViewFactory());

  // 4. 注册内置命令（保存、撤销、复制等）
  editor->initCommands();

  // 5. 注册交互状态（拖拽、选择、连线等）
  nged::addImGuiInteractions();

  // 6. 加载字体
  nged::ImGuiResource::reloadFonts();

  // 7. 创建新文档并打开默认视图
  editor->createNewDocAndDefaultViews();
}
```

**`defaultGraphItemFactory()`** 注册了：注释框（`comment`）、路由点（`router`）、箭头（`arrow`）、分组框（`group`）。

## 6. 类型检查与颜色提示

使用 `TypedNode` 代替 `Node` 可以自动进行类型检查：

```cpp
class TypedAddNode : public nged::TypedNode {
public:
  TypedAddNode(nged::Graph* parent)
    : TypedNode(parent, "add", "加法",
        {"float", "float"},  // 两个输入都是 float
        {"float"})           // 输出是 float
  {}
};
```

### 注册类型

```cpp
auto& ts = nged::TypeSystem::instance();
ts.registerType("float");
ts.registerType("int", "float");              // int 可向上转型为 float
ts.registerType("vec3", "", Color{255,0,0});  // vec3 带颜色提示（红色）
ts.setConvertable("int", "float", true);      // int → float 允许
```

类型兼容时，引脚和连线会显示该类型的颜色提示。

## 7. 子图支持

子图节点允许在节点内部包含另一个独立的图，双击即可进入编辑：

```cpp
class SubGraphNode : public nged::Node {
  nged::GraphPtr subgraph_;
public:
  SubGraphNode(nged::Graph* parent)
    : nged::Node(parent, "subgraph", "子图") {
    subgraph_ = std::make_shared<nged::Graph>(parent->docRoot(), parent, "子图");
  }

  // 关键：返回子图，编辑器由此识别为子图节点
  nged::Graph* asGraph() override { return subgraph_.get(); }

  // 序列化需要同时处理子图
  bool serialize(nged::Json& json) const override {
    return Node::serialize(json) && subgraph_->serialize(json);
  }
  bool deserialize(nged::Json const& json) override {
    return Node::deserialize(json) && subgraph_->deserialize(json);
  }

  // 重命名时同步子图名称（确保面包屑路径正确）
  bool rename(nged::String const& desired, nged::String& accepted) override {
    if (Node::rename(desired, accepted)) {
      subgraph_->rename(accepted);
      return true;
    }
    return false;
  }
};
```

- **双击进入**：双击子图节点进入子图编辑。
- **Shift + 双击**：在新视图窗口中打开子图。
- **面包屑路径**：菜单栏显示当前路径，可点击返回上层。
- **U 键**：快速回到父图。

## 9. 自定义交互状态

交互状态控制用户如何与画布交互。继承 `NamedInteractionState` 并实现关键方法：

```cpp
class MyState : public nged::NetworkView::NamedInteractionState<MyState> {
public:
  static constexpr nged::StringView className = "my-state";

  // 优先级：数字越小越先执行（0 最高，100 最低）
  int priority() const override { return 50; }

  // 是否应该进入此状态
  bool shouldEnter(nged::NetworkView const* view) const override {
    return ImGui::IsKeyDown(ImGuiKey_M) && view->isFocused();
  }

  // 进入状态
  void onEnter(nged::NetworkView* view) override {
    nged::MessageHub::info("进入自定义模式");
  }

  // 每帧更新（返回 true 阻止后续状态更新）
  bool update(nged::NetworkView* view) override {
    return false;
  }

  // 绘制
  void draw(nged::NetworkView* view) override {
    view->canvas()->drawText({0, 0}, "自定义模式中...");
  }

  // 是否应该退出
  bool shouldExit(nged::NetworkView const* view) const override {
    return ImGui::IsKeyReleased(ImGuiKey_M);
  }

  // 退出状态
  void onExit(nged::NetworkView* view) override {
    nged::MessageHub::info("退出自定义模式");
  }
};

// 注册（在 init 中调用）
MyState::registerInteraction<MyState>();
```

### 内置交互状态

| 状态 | 优先级 | 功能 |
|------|--------|------|
| `HandleView` | 0 | 平移（中键拖拽）、缩放（滚轮） |
| `AnimationState` | 10 | 平滑动画过渡 |
| `MoveState` | 10 | 拖拽移动图元 |
| `SelectionState` | 50 | 框选/点选 |
| `LinkState` | 50 | 拖拽连线 |
| `CreateNodeState` | 50 | 右键/Ctrl+P 创建节点 |
| `CutLinkState` | 50 | 按 Y 划过切断连线 |
| `EditArrow` | 2 | 编辑箭头 |
| `ResizeBoxState` | — | 调整注释框/分组框大小 |
| `HandleShortcut` | 100 | 快捷键检测 |
| `HandleContextMenu` | — | 右键菜单 |

## 10. 自定义视图

继承 `GraphView` 并实现 `onDocModified()` 和 `draw()`：

```cpp
class DataView : public nged::ImGuiGraphView<DataView, nged::GraphView> {
public:
  DataView(nged::NodeGraphEditor* editor, nged::NodeGraphDocPtr doc)
    : ImGuiGraphView(editor, doc) { kind_ = "data"; setTitle("数据视图"); }

  void onDocModified() override {}
  void onGraphModified() override {}

  void drawContent() {
    ImGui::Text("这里是自定义视图内容");
    ImGui::Text("绑定到图: %s", graph() ? graph()->name().c_str() : "无");
  }
};

// 注册自定义视图
class MyViewFactory : public nged::ViewFactory {
  nged::GraphViewPtr createView(
    nged::String const& kind,
    nged::NodeGraphEditor* editor,
    nged::NodeGraphDocPtr doc) const override
  {
    if (kind == "data")
      return std::make_shared<DataView>(editor, doc);
    return nged::defaultViewFactory()->createView(kind, editor, doc);
  }
};

// 设置
editor->setViewFactory(std::make_shared<MyViewFactory>());

// 添加打开此视图的命令
editor->commandManager().add(new nged::CommandManager::SimpleCommand{
  "View/Data", "打开数据视图",
  [](nged::GraphView* view, nged::StringView) {
    view->editor()->addView(view->doc(), "data");
  },
  nged::Shortcut{'D', nged::ModKey::CTRL},
  "*"
});
```

## 11. 事件系统

`GraphEventHub` 提供 Signals（广播）和 Requests（需全员同意）：

```cpp
auto& events = editor->events();

// 订阅事件
auto h1 = events.onItemSelected.connect([](nged::NetworkView* v, nged::GraphItem* item) {
  nged::MessageHub::infof("选中了: {}", item->asNode() ? item->asNode()->name() : "?");
});

auto h2 = events.afterPaste.connect([](nged::Graph* g, nged::GraphItem** items, size_t n) {
  nged::MessageHub::infof("粘贴了 {} 个图元", n);
});

auto h3 = events.onLinkSet.connect([](nged::Link* link) {
  nged::MessageHub::info("连线已建立");
});

// 取消订阅
events.onItemSelected.disconnect(h1);
```

### 所有可用事件

| 事件 | 类型 | 说明 |
|------|------|------|
| `requestAddItem` | Request | 添加图元前（返回 false 阻止） |
| `onItemAdded` | Signal | 图元已添加 |
| `requestRemoveItem` | Request | 移除图元前 |
| `onItemRemoved` | Signal | 图元已移除 |
| `onItemMoved` / `onItemModified` | Signal | 图元移动/修改 |
| `onItemClicked` / `onItemDoubleClicked` | Signal | 单击/双击图元 |
| `onItemHovered` / `onItemSelected` | Signal | 悬停/选中 |
| `onSelectionChanged` | Signal | 选择变化 |
| `requestLinkSet` | Request | 连线前 |
| `onLinkSet` / `onLinkRemoved` | Signal | 连线建立/移除 |
| `onNodeRenamed` | Signal | 节点重命名 |
| `afterPaste` | Signal | 粘贴后 |
| `beforeViewUpdate` / `afterViewUpdate` | Signal | 视图更新前后 |
| `beforeViewDraw` / `afterViewDraw` | Signal | 视图绘制前后 |

## 12. 保存与加载

NGED 内置了完整的保存/加载机制（JSON 格式，`.ng` 扩展名）：

```cpp
// 保存：Ctrl+S 自动调用
editor->saveDoc(doc);

// 另存为：Ctrl+Shift+S
editor->saveDocAs(doc, "/path/to/file.ng");

// 打开：Ctrl+O
editor->openDoc("/path/to/file.ng");

// 自定义文件扩展名
editor->setFileExt("mygraph");
```

### 无人工厂序列化

如果节点有额外数据需要保存，重写 `serialize`/`deserialize`：

```cpp
bool serialize(nged::Json& json) const override {
  if (!Node::serialize(json)) return false;
  json["myData"] = myCustomValue;
  return true;
}

bool deserialize(nged::Json const& json) override {
  if (!Node::deserialize(json)) return false;
  myCustomValue = json.value("myData", 0);
  return true;
}
```

## 13. 无头模式

不需要 UI 的情况下也可以使用图数据结构：

```cpp
auto itemFactory = nged::defaultGraphItemFactory();
auto nodeFactory = std::make_shared<MyNodeFactory>();
auto doc = std::make_shared<nged::NodeGraphDoc>(nodeFactory, itemFactory);
doc->makeRoot();

// 创建节点
auto a = doc->root()->createNode("add");
auto b = doc->root()->createNode("in");
auto c = doc->root()->createNode("out");

// 连线
doc->root()->setLink(b->id(), 0, a->id(), 0);
doc->root()->setLink(a->id(), 0, c->id(), 0);

// 遍历
nged::GraphTraverseResult result;
doc->root()->travelTopDown(result, c->id());
for (auto acc : result) {
  std::cout << "节点: " << acc->name() << std::endl;
}

// 保存
doc->saveTo("output.ng");
```

## 14. 常见问题

### ImGui ID 冲突

当同一作用域内出现相同标签的控件时，ImGui 会报 ID 冲突：

```cpp
// ❌ 错误：两个 MenuItem 显示相同文字
ImGui::MenuItem("subgraph");
ImGui::MenuItem("subgraph");

// ✅ 方法 1：使用 ## 后缀区分 ID（不显示）
ImGui::MenuItem("subgraph##1");
ImGui::MenuItem("subgraph##2");

// ✅ 方法 2：PushID/PopID
for (int i = 0; i < n; i++) {
  ImGui::PushID(i);
  ImGui::MenuItem("subgraph");
  ImGui::PopID();
}
```

### 中文输入

默认内嵌字体（Roboto）不支持中文。需要加载中文字体：

```cpp
nged::ImGuiResource::reloadFonts();  // 先加载内置字体
ImFontConfig cfg;
cfg.MergeMode = true;
ImGui::GetIO().Fonts->AddFontFromFileTTF(
  "/usr/share/fonts/TTF/MapleMono-CN-Regular.ttf", 0.0f, &cfg);
// 1.92 动态字体：size=0，无需指定 glyph range，图集自动增长
```

### 高 DPI 缩放

Raylib 后端已自动处理 DPI（`FLAG_WINDOW_HIGHDPI` + `ScaleAllSizes`）。如 UI 仍然太小，手动调整：

```cpp
ImGui::GetStyle().ScaleAllSizes(2.0f);  // 200% 缩放
```

### 自定义布局

```cpp
editor->setDefaultLayoutDesc(R"(
hsplit:
  data:3:hide_tab_bar
  vsplit:6
    hsplit:7
      network:5
      inspector:3:hide_tab_bar
    message:3:hide_tab_bar
)");
```

**布局语法**：
- `hsplit:` — 水平分割子窗口
- `vsplit:` — 垂直分割子窗口
- `name:weight` — 视图名称和权重（越大占比越多）
- `:hide_tab_bar` — 隐藏标签栏
