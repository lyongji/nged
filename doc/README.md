# 介绍

欢迎阅读 `nged`（节点图编辑器）文档。`nged` 是一个用于创建节点图编辑器的 C++ 库，适用于可视化编程、着色器编辑或数据流建模。

## 概览

该库提供了灵活的基础框架：
- **图数据结构**：管理节点、连线及其连接关系。
- **视图系统**：渲染和交互式操作图。
- **可扩展性**：创建自定义节点类型和交互方式。
- **撤销/重做**：内置历史管理。
- **样式**：可定制的视觉外观。
